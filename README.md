# Portfolio
This is a personal portfolio website built using Next.js with a fully static architecture for optimal performance and scalability.

The project explores an AI-driven development workflow:

Claude is used for documentation and content structuring
Stitch is used for UI/UX design
Antigravity assists with code generation and implementation

The entire development process is powered by prompt engineering, showcasing how modern AI tools can accelerate and enhance web development.

⚡ Fast, minimal, and optimized for deployment.
